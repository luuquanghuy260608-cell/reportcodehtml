LaTeX Study Guide

LaTeX is a powerful system for writing documents, especially scientific papers, reports, theses, and mathematical content. It is widely used in universities and research.

1. What Is LaTeX?

  • LaTeX is a system for writing documents in a clean and professional style.
  • It is especially strong in math, scientific writing, and long reports.
  • It is different from Word because you write commands and LaTeX handles the formatting.

Example: A short document with a title and a paragraph.

\documentclass{article}
\begin{document}
Hello world.
\end{document}

2. Why Use LaTeX?

  • It makes papers look neat and consistent.
  • It is excellent for equations such as $E = mc^2$.
  • It helps with sectioning, references, and bibliography automatically.
  • It is widely used in universities and research.

Example: A thesis or report can be formatted automatically with headings, numbering, and references.

3. Basic Structure of a LaTeX Document

\documentclass{article}
\usepackage[utf8]{inputenc}

\begin{document}
Hello, this is my first LaTeX document.
\end{document}

This is the simplest LaTeX file. The documentclass tells LaTeX what kind of document it is.

4. Main Commands

  • \section{...} creates a section.
  • \subsection{...} creates a subsection.
  • \textbf{...} makes text bold.
  • \textit{...} makes text italic.
  • \begin{itemize} ... \end{itemize} creates a bullet list.

Example: Create a section and bold text in a document.

\section{Introduction}
This is \textbf{important} text.

5. Writing Math in LaTeX

LaTeX is famous for mathematical expressions. It is one of the best tools for writing formulas clearly.

  • Use $...$ for inline math.
  • Use $$...$$ or \begin{equation} for display math.
  • Example: $x^2 + y^2 = z^2$
\[ E = mc^2 \]

$\int_a^b x^2 \, dx$

6. Common Math Symbols

  • \alpha, \beta, \gamma
  • \sum, \int, \frac
  • \leq, \geq, \neq
  • \sqrt{}, \cdot

Example: Write a formula for a quadratic equation.

$x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}$

7. Lists and Environments

\begin{itemize}
  \item First item
  \item Second item
\end{itemize}

\begin{enumerate}
  \item First step
  \item Second step
\end{enumerate}

These environments are useful for making notes, assignments, and reports.

8. Tables and Figures

  • Tables are created with tabular.
  • Figures are inserted with \includegraphics.
  • These are useful for reports, theses, and research papers.

Example: A simple table can be written like this.

\begin{tabular}{|c|c|}
\hline
A & B \\
\hline
1 & 2 \\
\hline
\end{tabular}

9. References and Citations

  • LaTeX can manage bibliography with bibtex or biber.
  • It is very useful for academic writing and citation styles.
  • It helps create references automatically and neatly.

10. Applications of LaTeX

  • Research papers
  • Theses and dissertations
  • Books and lecture notes
  • CVs and resumes
  • Slides and presentations

Example: Many students use LaTeX for homework reports and academic papers because the layout is professional.

11. Important Notes

  • LaTeX is best for structured and professional documents.
  • It may feel harder at first, but it becomes very powerful with practice.
  • It is especially useful for people who write many equations or academic papers.
