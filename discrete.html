Discrete Mathematics Study Guide

Discrete mathematics studies countable and separate structures. It is very important in logic, algorithms, and computer science.

1. What It Is

  • Discrete mathematics studies things that can be counted one by one.
  • It is different from continuous math, where values can change smoothly.
  • It is very important in logic, algorithms, databases, and computer science.

Example: Count the number of students in a class one by one rather than using a continuous measurement.

2. Basic Ideas

  • A set is a collection of distinct objects.
  • A subset is a set inside another set.
  • Union means put everything together.
  • Intersection means keep only the common parts.
  • Logic uses statements like AND, OR, and NOT.

Example: If P = {1, 2} and Q = {2, 3}, then P ∩ Q = {2}.

3. Set Theory

Set theory is the foundation of many discrete math topics. It helps us group and compare objects clearly.

  • Example: A = {1, 2, 3}
  • Example: B = {3, 4, 5}
  • Union: A ∪ B = {1, 2, 3, 4, 5}
  • Intersection: A ∩ B = {3}
  • Subset: {1, 2} ⊆ {1, 2, 3}

4. Permutations and Combinations

These are methods for counting possibilities.

  • Permutation: order matters.
  • Combination: order does not matter.
  • Example: choosing 1st and 2nd place is a permutation.
  • Example: choosing 2 students from 5 is a combination.
Permutation: P(5, 2) = 5 × 4 = 20
Combination: C(5, 2) = 10

5. Why It Matters in Computer Science

  • Used in algorithms and data structures.
  • Used in logic circuits and databases.
  • Used in probability and counting problems.
  • Helps you think step by step and solve problems clearly.

Example: A search algorithm uses logic and counting to find an item efficiently in a list.

6. Important Notes

  • Order matters in permutations but not in combinations.
  • Sets help us describe relationships between objects.
  • Discrete math is the logic behind many computer science ideas.
