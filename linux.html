Linux / Ubuntu Study Guide

Linux is an operating system widely used in servers, programming, and development environments. Ubuntu is one of the most popular Linux distributions.

1. What It Is

  • Linux is a free and open-source operating system.
  • Ubuntu is a beginner-friendly Linux distribution.
  • Linux is commonly used in programming, servers, cloud systems, and embedded devices.

Example: Open a terminal and type:

pwd
ls
cd Documents

2. Why Linux Is Useful

  • It is stable and powerful.
  • It is used by developers around the world.
  • It supports programming tools, terminals, and automation well.
  • It is often used in servers and remote computers.

Example: A developer can use Linux to run Python, install packages, and manage a server remotely.

3. Key Commands

  • pwd shows the current working directory.
  • ls lists files and folders.
  • cd changes the current directory.
  • mkdir creates a new folder.
  • rm removes files or folders.
  • cp copies files.
  • mv moves or renames files.
  • sudo runs a command with administrator privileges.
pwd
ls
cd Documents
mkdir test_folder
cp file.txt test_folder/
rm test_folder/file.txt

4. Example: Working with Files

mkdir myproject
cd myproject
nano hello.txt

This shows how a beginner can create a folder, enter it, and create a text file.

5. Git with Linux

Git is a version control system used to track changes in code. It works very well in Linux because developers often use the terminal for coding and collaboration.

  • git init starts a new Git repository.
  • git status shows the current state of files.
  • git add . stages files for commit.
  • git commit -m "message" saves a new version.
  • git push uploads changes to a remote repository.
  • git pull downloads the latest changes.
git init
git status
git add .
git commit -m "first commit"
git push

6. Why Git Matters

  • It helps you save versions of your project.
  • It makes teamwork easier.
  • It lets you go back to older versions if something breaks.
  • It is used everywhere in software development.

Example: If you break your code, you can return to an older Git version instead of losing your work.

7. More Useful Linux Commands

  • cat shows the contents of a file.
  • grep searches for text inside files.
  • chmod changes permissions of files.
  • whoami shows the current user.
  • top shows running processes.
whoami
cat hello.txt
grep "hello" file.txt
chmod +x script.sh
top

8. Important Notes

  • Be careful with rm -r because it deletes folders permanently.
  • Use man to read the manual of a command.
  • apt install is commonly used to install software on Ubuntu.
  • Use apt update before installing packages.
  • Linux is powerful because you can automate tasks with the terminal.

Example: Install Git on Ubuntu with:

sudo apt update
sudo apt install git

9. Docker Compose

Docker Compose is a tool that helps you run multiple containers with one configuration file. It is very useful when your project needs services such as a web app, database, and cache at the same time.

  • docker compose up starts all services defined in the Compose file.
  • docker compose down stops and removes the containers.
  • docker compose build builds or rebuilds the images.
  • docker compose ps shows the running containers.
version: "3.8"
services:
  web:
    image: nginx
    ports:
      - "8080:80"
  db:
    image: postgres

Example: Start the services with:

docker compose up -d
