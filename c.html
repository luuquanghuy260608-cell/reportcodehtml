C Programming Study Guide

C is a powerful and low-level programming language. It is commonly used in operating systems, embedded systems, and performance-critical applications.

1. What It Is

  • C gives direct control over memory and hardware.
  • It is fast and efficient.
  • It is used in operating systems, drivers, embedded systems, and compilers.

Example: A tiny C program that prints a message.

#include <stdio.h>
int main() {
    printf("Hello from C\n");
    return 0;
}

2. Key Concepts

  • Variables must have a declared type, such as int or char.
  • Input and output are handled with printf and scanf.
  • Loops such as for, while, and do while repeat code.
  • Functions help organize a program into small parts.
  • Pointers allow direct memory access.

Example: Use an if statement to check a number.

int x = 10;
if (x > 5) {
    printf("x is greater than 5\n");
}

3. Array

An array is a collection of elements of the same type stored in contiguous memory. It is useful when you want to store many values together.

  • Arrays have a fixed size.
  • Elements are accessed using an index, starting from 0.
  • Arrays are useful for storing lists of numbers, characters, or other data.
  • Common operations include reading values, updating values, and looping through elements.
#include <stdio.h>

int main() {
    int numbers[5] = {1, 2, 3, 4, 5};

    for (int i = 0; i < 5; i++) {
        printf("%d\n", numbers[i]);
    }

    return 0;
}

4. Pointers in C

A pointer is a variable that stores the address of another variable in memory. It is very useful when you want to work directly with memory.

  • * means “value at the address”.
  • & means “address of”.
  • A pointer helps pass large data to functions efficiently.
  • Pointers are used in arrays, strings, and linked lists.
#include <stdio.h>

int main() {
    int x = 10;
    int *ptr = &x;

    printf("Value of x: %d\n", x);
    printf("Address of x: %p\n", (void*)ptr);
    printf("Value pointed by ptr: %d\n", *ptr);

    return 0;
}

5. Memory and Dynamic Allocation

Memory is where data is stored while a program is running. In C, you can use stack memory and heap memory. Dynamic memory allocation lets you request memory at runtime.

  • malloc allocates memory dynamically.
  • free releases memory when it is no longer needed.
  • Using memory correctly helps avoid errors such as memory leaks.
  • Memory must be managed carefully in C.
#include <stdio.h>
#include <stdlib.h>

int main() {
    int *ptr = malloc(sizeof(int));
    if (ptr == NULL) {
        return 1;
    }

    *ptr = 42;
    printf("%d\n", *ptr);
    free(ptr);
    return 0;
}

6. References in C

In C, a pointer is the main way to work with references to memory locations. It lets you access and change data indirectly.

  • &x gives the address of variable x.
  • *ptr accesses the value stored at that address.
  • Pointers are used to pass variables by address to functions.
  • They are very important in arrays, strings, and dynamic memory.
#include <stdio.h>

void change(int *p) {
    *p = 100;
}

int main() {
    int x = 10;
    change(&x);
    printf("%d\n", x);
    return 0;
}

7. Applications of References in C

  • Used in function parameters to modify values.
  • Essential for dynamic memory allocation with malloc.
  • Used in data structures like linked lists and trees.
  • Helpful for efficient handling of large data.

Example: A function can change a value through a pointer.

void swap(int *a, int *b) {
    int temp = *a;
    *a = *b;
    *b = temp;
}

8. Important Notes

  • Always free memory allocated by malloc.
  • Do not use freed memory.
  • Common C algorithms include searching, sorting, and traversal.
  • Be careful with pointers because mistakes can crash the program.

9. File Handling

C allows you to read from and write to files using file pointers. This is useful for saving data permanently.

  • fopen opens a file.
  • fprintf writes data to a file.
  • fscanf reads data from a file.
  • fclose closes the file.
#include <stdio.h>

int main() {
    FILE *file = fopen("data.txt", "w");
    if (file != NULL) {
        fprintf(file, "Hello from C\n");
        fclose(file);
    }
    return 0;
}

10. Linked List

A linked list is a data structure made of nodes. Each node stores data and a pointer to the next node. It is useful when you need to insert or remove elements dynamically.

  • Each node contains data and a pointer.
  • The first node is called the head.
  • Linked lists are flexible, but they use more memory than arrays.
#include <stdio.h>
#include <stdlib.h>

struct Node {
    int data;
    struct Node *next;
};

int main() {
    struct Node *head = malloc(sizeof(struct Node));
    head->data = 10;
    head->next = NULL;

    printf("%d\n", head->data);
    free(head);
    return 0;
}

11. Important Algorithms to Know

  • Traversal: visit every node in the list.
  • Insertion: add a new node at the beginning or end.
  • Deletion: remove a node safely.
  • Searching: find an element in an array or linked list.

Example: Search for a number in an array using a loop.

int arr[] = {2, 4, 6, 8};
for (int i = 0; i < 4; i++) {
    if (arr[i] == 6) {
        printf("Found\n");
    }
}
